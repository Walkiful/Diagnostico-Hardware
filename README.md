# Diagnóstico de Hardware em Python

Um script simples e eficiente que coleta informações do sistema — como CPU, memória, disco e rede — e gera relatórios completos em **.txt** e **.html**.

Ideal para técnicos de hardware, estudantes de tecnologia e entusiastas que desejam automatizar a análise de desempenho do computador.

### Instale o Python
Baixe em [python.org/downloads](https://www.python.org/downloads/)  

### Instale as dependências
Abra o terminal na pasta do projeto e rode:
